### 托管到 Github

- 创建仓库 | Create a new repository
- 本地项目推送到远程服务器

```bash
git init
git add .
git commit -m "my blog first commit"
git remote add origin git@github.com:【你的 github 名字】/【你的 repository 名字】.git
git push -u origin master
```
